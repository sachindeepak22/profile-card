Tailwind Gradient Quick Notes

Linear, radial, conic (prism), and glassmorphism patterns

1) Linear Gradient

bg-gradient-to-r from-indigo-600 via-purple-600 to-blue-500

2) Radial Gradient

bg-[radial-gradient(circle_at_top_left,_#818cf8,_#1e1b4b)]

3) Conic / Prism Style

bg-[conic-gradient(from_180deg_at_50%_50%,_#6366f1,_#a855f7,_#06b6d4,_#6366f1)]

4) Glassmorphism Card

bg-white/20 backdrop-blur-md border border-white/30 shadow-xl rounded-2xl

5) Gradient Text

bg-gradient-to-r from-indigo-600 to-blue-500 bg-clip-text text-transparent

6) Gradient Button

text-white bg-gradient-to-r from-indigo-600 to-blue-500 hover:from-indigo-500 hover:to-blue-400

Rule

Neutral base + single accent keeps UI clean and professional.
